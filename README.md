## How to use 🫁

**requirements:**
  - docker >= 27.3.1
  - docker compose >= 1.29.2

**1 - Clone this repo:**
```
git clone git@github.com:upALX/RegistraAI-Back-end.git
```
**2 - Build the aplication and run (first time)**
```
docker-coompose up --build
```

**3 - Disclaimer: if you need only run:** 

```
docker-compose up
```
